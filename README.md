# **Catch the Square**  
**Zahraj si na mém portfoliu:** [imunegh.github.io/portfolio](https://imunegh.github.io/portfolio/)

### **CZ Popis**  
Jedná se o jednoduchou klikačku, kde se zvedá skóré za každou trefenou kostičku. Když kostka spadne na zem, hra končí.

### **EN Description**  
This is a simple clicking game, where the score goes up for every target hit with your mouse. When the cube drops, the game ends.
